# Chalk — AI Math Visualization Tool

## Project Context

This is a hackathon project for the "Built with Opus 4.6" Claude Code Hackathon (Feb 10-16, 2026).
Builder: Ani Potts (solo). Goal: Win.

## Key Files

- CHALK_SPEC.md — Research spec and product requirements
- AGENT_TEAM_PROMPT.md — Agent team coordination instructions
- BATTLE_PLAN.md — Master hackathon plan (generated by agent team)
- COMPONENT_SCHEMAS.md — Zod schemas for viz components
- SYSTEM_PROMPT.md — AI prompt design for viz generation
- COMPETITIVE_ANALYSIS.md — Hackathon competitive landscape
- DEMO_SCRIPT.md — 3-minute demo script
- TECH_DECISIONS.md — Architecture decisions with reasoning
- RESEARCH_FINDINGS.md — Raw research and key links

## Tech Stack

Next.js 15, TypeScript, Tailwind, json-render, Mafs, React Three Fiber, D3, KaTeX, mathjs, Supabase, Vercel

## Commands

- `npm run dev` — local dev server
- `npm run build` — production build
- `npm test` — run tests
